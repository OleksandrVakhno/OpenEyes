About the App
Globally, at least 2.2 billion people have a vision impairment or blindness. (Source: WHO).

Discovering these numbers, we decided to create a way for the visually impaired to hear what we see.

Using Keras (a deep learning library), we created a recurrent neural network for automatic caption generations. After Camera permissions have been granted from the front end, this cross-platform application sends the image to our RESTful API (created using flask) that returns a spoken caption to the user.
